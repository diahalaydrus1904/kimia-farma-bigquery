# Kimia Farma BigQuery Analysis

This repository contains SQL scripts used to:
- Import Kimia Farma datasets into BigQuery
- Create an analysis table based on transaction, product, inventory, and branch data

## Tables Used
- kf_final_transaction
- kf_inventory
- kf_kantor_cabang
- kf_product

## Output Table
- kf_analysis

## Tools
- Google BigQuery
- SQL
